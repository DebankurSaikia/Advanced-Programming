# MyFirstRNApp

A React Native application that functions as a digital counter with theme switching capabilities.

## Features

- Increment counter value
- Decrement counter value
- Prevent negative counter values
- Reset counter to zero
- Toggle between Light Mode and Dark Mode
- Responsive layout using Flexbox
- Dynamic UI updates using React state management

## Technologies Used

- React Native
- TypeScript
- React Hooks (`useState`)

## Project Structure

```text
MyFirstRNApp/
├── App.tsx
├── package.json
├── android/
├── ios/
└── ...
```

## Installation

Install dependencies:

```bash
npm install
```

## Run the Application

### Start Metro

```bash
npm start
```

### Run on Android

```bash
npm run android
```

## Implementation Details

- State management using `useState`
- Counter increment, decrement, and reset functionality
- Validation to prevent negative values
- Dynamic theme toggling
- Conditional styling for Light and Dark modes
- Clean UI layout using Flexbox
- Responsive user interface using React Native components
