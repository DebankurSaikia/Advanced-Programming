# Course Enrollment Dashboard

A React application developed as part of **Assignment 08** for the **Advanced Programming** course.

## Assignment Overview

Develop a course enrollment dashboard that manages student records and course enrollments using React state management and modern JavaScript data structures.

## Features

- Add new students
- Remove students by ID
- Display students sorted by GPA (descending)
- Display all unique enrolled courses
- Filter students by course
- Dynamic state updates without direct mutation
- Course management using `Set`
- Student mapping using `Map`

## Technologies Used

- React
- TypeScript
- Vite
- React Hooks (`useState`)

## How to Run

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

## Assignment Requirements Implemented

- State management using `useState`
- Student data management
- Add and remove student functionality
- GPA-based sorting
- Course-based filtering
- Unique course extraction using `Set`
- Student lookup using `Map`
- Use of `map`, `filter`, and `reduce`
- Immutable state updates using the spread operator
