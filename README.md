# Advanced Programming

This repository contains assignments completed as part of the **Advanced Programming(CSBT215)** course.

## Repository Organization

Each assignment is maintained in a **separate Git branch** as required for submission.

The `main` branch serves as an index and navigation page for the repository. To view a specific assignment, select the corresponding branch from the GitHub branch dropdown menu.

## Structure of Each Assignment Branch

Each assignment branch contains:

- **README.md** – Assignment question, objective, and assignment-specific details.
- **AssignmentX_CSB24008/** – Main assignment folder containing the implementation and supporting files.

Depending on the assignment, the assignment folder may contain:

- **Source-Code/** – Source code files (C, Java, Python, etc.).
- **Project Folder** – React or React Native project files (e.g., Todo Application, Course Enrollment Dashboard, Digital Counter App).
- **Documentation/** – Time and space complexity analysis documents (where applicable).
- **Sample-Output/** – Screenshots, screen recordings, sample outputs, execution results, and release APKs (where applicable).
